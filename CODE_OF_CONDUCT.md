# Autohand Code Agent SDK for Java Code of Conduct

The Autohand community includes people with different backgrounds, skill levels,
languages, time zones, and constraints. That variety is a strength, and it also
means we need a shared baseline for how we work together.

This code applies to spaces managed by the Autohand Java SDK project and
Autohand AI, including GitHub issues, pull requests, discussions, community
channels, and project events. Violations outside those spaces may affect a
person's ability to participate in them.

If you believe someone is violating this code of conduct, report it by emailing
[hey@autohand.ai](mailto:hey@autohand.ai).

- **Be friendly and patient.** Good open-source work often happens across async
  conversations and incomplete context.
- **Be welcoming.** We strive to support people of all backgrounds and identities,
  including but not limited to race, ethnicity, culture, national origin, color,
  immigration status, social and economic class, education level, sex, sexual
  orientation, gender identity and expression, age, size, family status,
  political belief, religion, and mental or physical ability.
- **Be considerate.** Your work will be used by other people, and you will depend
  on the work of others. Think about the impact of API, documentation, and
  behavior changes before proposing them.
- **Be respectful.** Disagreement is expected. Personal attacks, insults, and
  intimidation are not.
- **Choose words carefully.** Harassment and exclusionary behavior are not
  acceptable, including violent threats, discriminatory language, posting sexual
  or violent material, doxing, personal insults, unwelcome sexual attention, or
  encouraging any of those behaviors.
- **When we disagree, try to understand why.** Technical disagreements are best
  handled by grounding discussion in examples, tests, compatibility, and the
  needs of people building with the SDK.

Original text adapted from the Autohand Code CLI code of conduct and the
[Speak Up! project](http://web.archive.org/web/20141109123859/http://speakup.io/coc.html).
